# RiskClause AI

RiskClause AI is an intelligent tool that analyzes contracts to detect potentially risky clauses. It empowers users by highlighting contract terms that may pose legal or business risks, making contract review faster, clearer, and more accurate.

---

## Features

- Automated Risk Detection: Quickly identify risky clauses in contracts using AI.  
- Clear Explanations: Understand why a clause is flagged as risky with detailed insights.  
- Risk Scoring: Get an overall risk assessment for the entire contract.  
- User-Friendly Interface: Simple and intuitive design for easy contract review.

---

## Installation

1. Clone the repository:  
   `git clone https://github.com/chirathdinduwara/RiskClauseAI.git`

2. Navigate to the project directory:  
   `cd riskclause-ai`

3. Create a `.env` file in the root directory to store environment variables. For example:
    PORT=3003
    GEMINI_API_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx


4. Install dependencies:  
    `npm install`

5. Start the application:  
    `npm start`

---

## Usage

- Paste the contract text into the interface.  
- The AI analyzes the text and highlights any risky clauses it detects.  
- Review detailed explanations and risk scores for better understanding.  

---

## Technologies Used

- Frontend: React, Tailwind CSS  
- Backend: Node.js, Express.js  
- AI Integration: Google Gemini AI API
